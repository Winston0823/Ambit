Why this looks right

Direct continuity with the discovery card

  • Same gradient direction (top → bottom, tan → coffee) on every tan surface — pinned tiles, regular avatars, and the unanswered-reach-out cards. The avatar is literally a 50pt thumbnail of the full discovery card.
  • Same chip language (pill, 1.5pt terracotta border, no fill) for status pins — "Reply" / "Hired" / countdown. No invented badges.
  • Same cream-pill-with-dot pattern as the "● 2 SHARED · USC" badge on the card — used here for the urgent countdown when a reach-out is < 24h from expiry.
  • Same chrome — "ambit" Fraunces wordmark center, dark rgb(41,41,41) nav bar at bottom, white canvas. The inbox sits inside the existing app frame instead of inventing a new one.

Hierarchy via material, not chrome

  • Unanswered reach-outs become mini cards — the only surfaces in the list that get the tan-gradient treatment, which is the brand's signal-of-presence. The eye goes there first.
  • Active conversations are flat rows divided by hairlines. Quiet. Easy to scan.
  • Auto-closed are the same row, just at 50% opacity with a desaturated avatar. Still visible if you want to re-reach-out, but visually dismissed.
  • Pinned live above the list as a horizontal strip — tiny tan-gradient tiles with name, amber unread dot, sage live dot.

The countdown

  • On the pending mini card, the countdown is a solid terracotta chip (matches "matched skill" chip language) when < 24h left — high contrast against the tan card surface.
  • When ≥ 24h left, it drops to an outline chip (matches non-matched skill chip). Same visual vocabulary, just downgraded.
  • Format: 2d 14h left → 21h to reply → 14m to reply.

Behavior summary

  • Pin via long-press on any row (max 4 — iMessage parity).
  • One flat list, no "Today / Yesterday" grouping. Sort: pending reach-outs → last_message_at desc → auto-closed at the bottom.
  • Last message preview always shown, prefixed with "You:" or partner name so it's instantly clear who owes a reply.
  • Auto-close at 72h → both users re-eligible in each other's discovery feed with a "missed reach out" ribbon on the card. Re-reach-out resets the timer and inserts a system message.
