# Ambit — iOS

Native iOS port of Ambit, a chat-first proximity-based hiring app for Bay Area startups. SwiftUI, iOS 26+, Liquid Glass UI.

## Project setup

This project is generated by [XcodeGen](https://github.com/yonaskolb/XcodeGen) from `project.yml`. To open in Xcode:

```bash
brew install xcodegen
xcodegen generate
open Ambit.xcodeproj
```

## Structure

```
Sources/Ambit/
  AmbitApp.swift           App entry
  RootView.swift           Top-level container
  Theme/                   Design tokens (palette, typography)
  Components/              Reusable views (LiquidNavBar, etc.)
  Screens/                 Discovery, Chat, Projects, Profile
  Resources/
    Fonts/                 Bundled fonts (Zodiak Bold, Plus Jakarta Sans)
    Assets.xcassets        App icon, accent color
    Info.plist             Generated from project.yml
```

## Design language

Warm-tan + cream palette. Zodiak Bold for display, Plus Jakarta Sans for body. Floating liquid-glass nav bar with Discovery / Chat / Projects / Profile.
