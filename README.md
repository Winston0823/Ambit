# Ambit — iOS

Native iOS port of Ambit, a chat-first, proximity-based **student collaboration platform**. Connects student project owners with student seekers across Bay Area campuses. SwiftUI, iOS 26+, Liquid Glass UI.

> First-layer scaffold. Real auth, chat (Stream), and feed APIs (Supabase) layer in next. See `/Users/winstypoo/Ambit-iOS-docs/spec-notes.md` for the engineering brief distilled from Ambit Product Specification v1.0.

## Project setup

The Xcode project is generated by [XcodeGen](https://github.com/yonaskolb/XcodeGen) from `project.yml`. After cloning:

```bash
brew install xcodegen
xcodegen generate
sudo xcodebuild -license accept   # first-time only
open Ambit.xcodeproj
```

Then ⌘R to run on the iOS 26 simulator.

## Structure

```
Sources/Ambit/
  AmbitApp.swift           App entry — registers fonts then shows RootView
  RootView.swift           Top-level container — hosts active screen + floating nav
  Theme/
    Brand.swift            Color tokens (warm tan + cream + ink scale)
    Typography.swift       Type ramp (Zodiak Bold display, Plus Jakarta body)
    FontLoader.swift       Registers bundled OTFs at launch via CoreText
  Components/
    LiquidNavBar.swift     iOS 26 .glassEffect() bar w/ animated warm-tan pill
  Screens/                 Discover · Chats · Notifications · Profile (stubs)
  Resources/
    Fonts/                 Zodiak-Bold.otf, PlusJakartaSans-Regular.otf
    Assets.xcassets        AppIcon (logo exported from Figma)
    Info.plist             Generated from project.yml
```

## Tab structure

Per spec § 5.1.1, **Seeker** view:

| Tab 1 | Tab 2 | Tab 3 | Tab 4 |
|---|---|---|---|
| Discover | Chats | Notifications | Profile |

Owner-side variant (Candidates / Inbox / Chats / Profile) is wired up when role-switching ships on the Profile screen.

## Design language

Warm-tan + cream palette. Zodiak Bold for display, Plus Jakarta Sans for body. Floating liquid-glass nav bar with a warm-tan animated pill that morphs to the active tab. Sourced from the Figma file `AMBIT` (Style And References page).
